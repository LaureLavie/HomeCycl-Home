# HomeCycl'Home

HomeCycl'Home est une application web dédiée à l’univers du vélo et du cyclisme à domicile.

## À propos

Ce dépôt contient le code source du projet **HomeCycl'Home**, développé principalement en **TypeScript** avec une partie en **JavaScript**.

## Fonctionnalités

- Interface web moderne
- Base de code majoritairement en TypeScript
- Projet orienté expérience utilisateur autour du cyclisme à domicile

## Technologies

- TypeScript
- JavaScript

## Installation

```bash
npm install
```

## Lancement en développement

```bash
npm run dev
```

## Build

```bash
npm run build
```
